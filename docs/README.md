## Quy ước

- Mỗi user story = 1 file `.md` riêng biệt
- Folder: `docs/{platform}/{section-key}/{ten-story}.md`
- File naming: kebab-case ASCII từ tiêu đề story (bỏ prefix `[GSA]`/`[AGA]`)
- Stories chưa có content: ghi rõ placeholder `> _Chưa có nội dung_`
